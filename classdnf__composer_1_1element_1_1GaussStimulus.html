Detailed Description

Localized Gaussian input stimulus applied to a neural field.

Provides a spatially localised, time-invariant (constant) input that models an external sensory cue or manually specified target location. The amplitude can be changed at runtime via setParameters() to simulate stimulus onset and offset.
