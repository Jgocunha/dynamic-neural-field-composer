Detailed Description

Persistent spatial memory that accumulates and decays field activity.

A MemoryTrace integrates above-threshold input from a NeuralField over time: while input exceeds threshold the trace grows with time constant tauBuild; below threshold it decays with time constant tauDecay. When fed back through a GaussKernel into the originating field, the trace sustains a localized peak even after the original stimulus is removed — implementing spatial working memory.
