Detailed Description

Damped-cosine convolution kernel for oscillatory field dynamics.

The kernel shape is a decaying cosine: amplitude * exp(-decay * |x|) * cos(2π * zeroCrossings * x). This produces alternating excitatory and inhibitory bands at increasing distances, which can generate wave-like or rhythmic activity patterns in a neural field.

When OscillatoryKernelParameters::outputFieldDimensions is set, the convolution result is resampled to the specified output size, enabling connections between fields of different spatial dimensions.
